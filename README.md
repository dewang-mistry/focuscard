# FocusCard

The public website for **FocusCard** — an iOS focus app that turns your iOS Reminders into a digital index card. One task, huge, on a black screen, mirrored to your Lock Screen.

Built for **deep work**, **ADHD focus**, and anyone who's ever propped a sticky note next to their laptop.
